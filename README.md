# Music serveer

This is a music server API that works as a fully fledged audio streaming/downloading platform backend.
## Features

- Multiple audio qualities - You can choose between 2 audio quality.
- Multiple platform - Custom search that combines the result of multiple platforms (Deezer/Qobuz). Also has an algorithm which avoids duplicate files download.
- User support - Users can register and login to their account.
- Playlist support.
- Song cache - each track gets cached when played for the first time.
